# CREDIT CARD FRAUD DETECTION - MODEL COMPARISON RESULTS

## Dataset:

Total Transactions : 284,807
Fraud Cases        : 492
Legitimate Cases   : 284,315

## Evaluation Metrics:

Precision = Correct fraud predictions / All fraud predictions
Recall    = Fraud caught / Total fraud cases
F1 Score  = Balance of Precision and Recall
AUC       = Ranking performance

## Final Results:

1. Logistic Regression

---

Precision : 0.681
Recall    : 0.838
F1 Score  : 0.752
AUC       : 0.967

Confusion Matrix:
TN = 42619
FP = 29
FN = 12
TP = 62

Insight:
High fraud detection rate (recall), suitable when catching fraud is priority.

## Final Ranking:

Best Recall     : Logistic Regression / XGBoost
Best Precision  : Autoencoder
Best F1 Score   : Autoencoder
Best AUC        : Random Forest
Best Balance    : Autoencoder

## Final Conclusion:

Autoencoder and Random Forest delivered the strongest practical results.
Logistic Regression and XGBoost are strong when maximizing fraud capture is the main objective.
